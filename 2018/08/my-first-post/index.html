My First Post

August 19, 2018 in category, subcategory

最近將服務都移過去 linode 上面，因為受限於選擇的方案儲存空間不夠，所以把腦筋動到 .github.io 上面 …

之前有玩過一些 static site generator，像是

  • python base: pelican
  • nodejs base: hexo

不過最近發現滿多人在用 go base 的 hugo，於是就來嘗試看看。

Github Page

這次的目標是要把 blog 架設到 https://mkfsn.github.io/blog/ ，換句話說是要部署到 blog 這個 repo 的 project page，所以就先來建立 local 的 repo.

Repository Branches

因為 github 預設的 branch 是 master (當然可以改，但還是需要一個介紹的 branch)， 所以我打算開一些 branch 做不同的用途：

  • master: 如同剛剛說的，就只是放 README.md 用的
  • hugo: 放 hugo 的 code 除了 public 資料夾
  • gh-pages: 這就是要放 generated code 的 branch，所以只放 public 資料夾的東西

master branch 就不多做描述，先切到 hugo branch 來開始正題吧。
